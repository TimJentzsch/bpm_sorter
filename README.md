# BPM Sorter

Sort Spotify playlists by their BPM.

## Config

```json
{
  "sourcePlaylist": "<playlist_url>",
  "minBpm": 65,
  "maxBpm": 135,
  "targets": [{
      "playlist": "<playlist_url>",
      "maxBpm": 90
    },
    {
      "playlist": "<playlist_url>",
      "maxBpm": 105
    },
    {
      "playlist": "<playlist_url>"
    }
  ]
}
```
